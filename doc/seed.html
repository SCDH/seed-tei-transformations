"att.transformable" is an attribute class that provides attributes for linking to transformation scenarios. Provides links to oXygen transformation scenarios provided in scenario or xpr files. The fragment identifier is evaluated as a scenario name. Provides links to Ant transformations provided in Ant build files. The fragment identifier is evaluated as a target name. Provides links to transformations declared in by some extension. Points to an <egXML> into which the current code example is wrapped into so that it results to a well formed TEI document that can be transformed.
